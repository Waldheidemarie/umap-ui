The API for the UMAP Rest Service is as follows.

Abbreviations:

  • HOST - the SAP Target Host which can be set in Window -> Preferences -> Umal 2 Abap Preferences
    The URI has to start with protocol://domain//furtherPath/umap

Service Definition:

GET HOST/umap
is the main Service which will do nothing in the moment it, it will return an XML list of available Services in the form:
This Service can be called as a Test in the Preference Page.

GET HOST/umap/transportRequests
Returns a XML List of available Workbench Transport Requests for the logged in user or an empty List.
Data Structure:
<?xml version="1.0" encoding="utf-8"?>
<requestlist>
<request id="request number">
<owner> Owner Name </owner>
<name> Request Name </name>
</request>
</requestlist>

GET HOST/umap/developmentClass?namespace=value
Returns an XML List of Customer Development Classes or an empty list.
Parameter Description:
namespace = value - represents an search string to restriced the seleced development classes, this parameter is optional. Not passing this parameter may result in a long runtime and a possible timeout
Example:
namespace=ZPL Data Structure:
